# Folio frontend

React + Vite + TypeScript SPA for Folio. See the [repo root README](../README.md) for setup and the [architecture doc](../docs/architecture.md) for how this talks to the backend.

```bash
npm install
npm run dev     # dev server at :5173, proxies /api to :8000
npm run build   # outputs dist/, served by the FastAPI backend
```
